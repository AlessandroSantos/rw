rwInputText
directive in module rwDirectives

Description

Componente de inputText padrão da arquitetura Rw. Este input já possui a validação do campo com suas respectivas mensagens.

Usage

as element:
<rw-input-text
       label="{string}"
       placeholder="{string}"
       restrict="{string}"
       ng-model="{string}"
       ng-required="{string}"
       ng-minlenght="{string}"
       ng-maxlenght="{string}">
</rw-input-text>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
labelstring

Texto a ser exibido na parte superior do input

placeholderstring

Texto a ser exibido dentro do input

restrictstring

Informa o tipo da diretiva (atributo, elemento, classe ...)

ngModelstring

Objeto do controller que será passado para a diretiva

ngRequiredstring

Informa se o input será obrigatório ou não

ngMinlenghtstring

Tamanho mínimo de caracteres do input

ngMaxlenghtstring

Tamanho máximo de caracteres do input
